Vad vi gör

Vi bygger en liten webbsida i JavaScript och CSS, och övar på att:

  • klona ett repo
  • göra ändringar i kod
  • commit:a och pusha till GitHub
  • skapa och arbeta i branches

Workshop-övningar

  1. Skapa en ny branch med ett pedagogiskt namn.
  2. Lägg till en text eller ändring i filerna.
  3. Commit:a med tydligt meddelande.
  4. Pusha till GitHub och skapa en pull request.
  5. Granska varandras PR:er och mergea tillsammans.

Javascript gör att sånt här händer

Skriv något kul här och se det i loggen nedan. Det kommer dock att försvinna då vi laddar om sidan.

Här kommer dina anteckningar att hamna.

Tips! Detta är nice!

  • Gör små, tydliga commits och skriv vad ändringen gjorde.
  • Använd branches för varje övning så ni kan experimentera utan att påverka main.
  • Be någon annan reviewa din pull request innan merge.
